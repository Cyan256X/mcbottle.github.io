# 网站预览说明

## 🌟 主要特性

### 1. 蓝色系设计
- **主色调**: 深蓝色 (#1e3a8a)
- **渐变背景**: 从浅灰到浅蓝的渐变
- **卡片设计**: 带阴影的白色卡片
- **响应式布局**: 适配各种屏幕尺寸

### 2. 导航栏
- 固定定位，滚动时半透明
- 包含5个主要部分：首页、项目、博客、关于、联系
- 鼠标悬停效果

### 3. 英雄区域
- 圆形头像占位符
- 个人介绍文字
- 社交链接图标 (GitHub, Twitter, LinkedIn, Email)

### 4. 项目展示
- 卡片式布局 (3个项目)
- 每个项目包含：
  - 项目图标和标题
  - 项目描述
  - 技术标签
  - 查看链接

### 5. 技术博客
- 文章列表 (3篇示例文章)
- 每篇文章包含：
  - 标题
  - 发布日期和分类
  - 文章摘要
  - 阅读全文链接

### 6. 侧边栏
- **关于我**: 个人简介
- **技能栈**: 技能进度条展示
- **联系我**: 社交链接
- **最新动态**: 更新记录

### 7. 页脚
- 四栏布局
- 网站信息
- 快速链接
- 项目资源
- 技术栈说明

## 🎨 设计亮点

### 色彩搭配
```
主色: #1e3a8a (深蓝)
辅色: #3b82f6 (蓝色)
强调色: #60a5fa (浅蓝)
背景: #f9fafb (浅灰)
文字: #1f2937 (深灰)
```

### 动画效果
1. **页面加载动画**: 元素渐入效果
2. **卡片悬停**: 上浮和阴影加深
3. **导航栏滚动**: 背景透明度变化
4. **平滑滚动**: 锚点链接平滑滚动

### 响应式设计
- **桌面端**: 两栏布局 (2fr 1fr)
- **平板端**: 自适应布局
- **移动端**: 单栏布局，隐藏导航菜单

## 📱 移动端适配

### 断点设计
```css
/* 桌面端: ≥ 768px */
.main-content {
  grid-template-columns: 2fr 1fr;
}

/* 移动端: < 768px */
@media (max-width: 768px) {
  .main-content {
    grid-template-columns: 1fr;
  }
  .nav-links {
    display: none;
  }
}
```

### 触摸优化
- 按钮和链接有足够的触摸区域
- 悬停效果在移动端有替代交互
- 字体大小适配移动设备

## 🚀 部署说明

### GitHub Pages
1. 推送到 `main` 分支
2. 自动部署到 `https://cyan256x.github.io`
3. 支持自定义域名

### 本地预览
```bash
# 使用 Python 简单 HTTP 服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

## 📝 自定义建议

### 内容更新
1. **替换头像**: 修改 `.avatar` 中的图标或添加图片
2. **更新项目**: 修改 `projects-grid` 中的项目信息
3. **添加文章**: 在 `#blog` 部分添加新的文章
4. **更新技能**: 修改技能进度条的宽度

1. **修改颜色**: 更新 CSS 变量中的颜色值
2. **调整布局**: 修改 Grid 和 Flexbox 布局
